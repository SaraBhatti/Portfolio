Cricket Analysis

Project Description

For this project, I employed Python and Excel to extract data from the ESPN website.

I scraped information from various tables and merged the relevant data using the string matching algorithm (FuzzyWuzzy) to match the correct player information.

Subsequently, I conducted an exploratory analysis on the dataset, creating visuals for my presentation.

I used Random Forest to create machine learning models. I then developed two Streamlit applications: one predicts match winners based on teams and playing grounds, while the other forecasts a batter's runs based on past performance
